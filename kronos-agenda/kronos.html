O projeto

O que é o Krónos?

Krónos é uma aplicação web de agenda pessoal com suporte a múltiplos usuários. Cada pessoa cria sua conta, faz login e gerencia seu próprio calendário de tarefas — sem interferência de outros usuários. O nome vem de Kronos, o deus grego do tempo.

🗓

Calendário visual

Interface de calendário mensal onde você clica em qualquer dia para ver, adicionar, concluir e excluir tarefas. As tarefas aparecem como pequenos blocos dentro de cada dia.

🔐

Multi-usuário

Cada usuário tem sua própria conta com login e senha. As tarefas são completamente isoladas — nenhum usuário acessa os dados de outro.

⚙️

Configurações

Cada usuário pode mudar sua senha ou apagar sua conta a qualquer momento pela página de configurações, com validações de segurança.

💾

Dados persistentes

Todos os dados ficam salvos no localStorage do navegador. As tarefas permanecem mesmo após fechar e reabrir o navegador.

O que ele faz

Funcionalidades

Tudo que foi construído do zero, linha por linha.

01

Cadastro de conta

Criação de usuário e senha com validações — campos obrigatórios, senha mínima de 6 caracteres e verificação de usuário duplicado.

02

Login com sessão

Autenticação por usuário e senha. A sessão é mantida no localStorage e protege todas as páginas de acesso não autorizado.

03

Calendário mensal

Grade de 7 colunas gerada dinamicamente pelo JavaScript com navegação entre meses e destaque automático no dia atual.

04

Tarefas por dia

Clique em qualquer dia para abrir o modal e adicionar tarefas. Suporta marcar como concluída, excluir e confirmar com Enter.

05

Isolamento de dados

As tarefas de cada usuário são salvas com uma chave única no localStorage, garantindo que nenhum dado seja compartilhado entre contas.

06

Gerenciamento de conta

Página de configurações com troca de senha (validando a senha atual) e opção de apagar a conta com todas as tarefas associadas.

O que foi aprendido

Cada funcionalidade do Krónos foi construída do zero durante o aprendizado de desenvolvimento front-end.

→Estrutura semântica de páginas HTML com formulários, inputs e navegação entre arquivos
→Estilização com CSS moderno, variáveis, Flexbox, Grid e animações
→Manipulação do DOM com JavaScript — criação dinâmica de elementos
→Eventos de formulário, clique e teclado com addEventListener
→Persistência de dados com localStorage, JSON.stringify e JSON.parse
→Métodos de array — .find(), .filter(), .map(), .push()
→Lógica de autenticação, sessão e proteção de rotas
→Separação de responsabilidades — HTML, CSS e JS em arquivos distintos
→Versionamento com Git — commits semânticos e push para GitHub
→Canvas API para renderização de gráficos animados em tempo real
